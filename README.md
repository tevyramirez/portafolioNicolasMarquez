# Portafolio - Nicolás Márquez

Portafolio web personal con múltiples estilos temáticos, construido con [Astro](https://astro.build).

## Estilos disponibles

- Atelier · Bravo Ortega · Datametrik Dark · Durán · Folio · Harvard · Madrid · Observatory · Swiss · Symmes

## Comandos

| Comando | Acción |
|---------|--------|
| `npm install` | Instalar dependencias |
| `npm run dev` | Iniciar servidor local en `localhost:4321` |
| `npm run build` | Build producción en `./dist/` |
| `npm run preview` | Vista previa del build |
