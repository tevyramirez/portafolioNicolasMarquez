## 💼 Caso de Éxito Destacado (Proyecto Privado)

### 📊 Plataforma SaaS: Sistema de Gestión de Cobros & Gastos Comunes (Aviso de Cobro)

*Un sistema integral de administración residencial y conciliación financiera en producción.*

- **El Desafío:** Digitalizar y automatizar el flujo completo de cobros, reportes de gastos comunes y conciliaciones para comunidades residenciales de forma robusta y sin fallos.
- **Solución Técnica:**
  - **Backend:** API REST robusta construida con **Django REST Framework** y autenticación segura con **JWT**. Procesamiento masivo de datos financieros y carga de planillas con **Pandas** y **Openpyxl**. Generación automatizada de reportes PDF dinámicos para los residentes mediante **WeasyPrint** y **ReportLab**.
  - **Frontend:** Panel de administración rápido e interactivo desarrollado con **React, Vite, Chakra UI y TanStack Query** para sincronización de datos en tiempo real.
  - **Calidad de Código:** Cobertura de tests unitarios y de integración con **Vitest**.
  - **Infraestructura:** Entorno completamente contenedorizado con **Docker** y **Docker Compose** para un despliegue ágil en producción.
